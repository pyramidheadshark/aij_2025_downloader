# AI Journey 2025 Downloader

Инструмент для автоматического парсинга, скачивания, сжатия и организации видеолекций с конференции AI Journey.

Скрипт умеет:
- Обходить защиту плеера через эмуляцию браузера (Playwright).
- Скачивать видеопотоки (HLS) с поддержкой докачки и восстановления битых сегментов.
- Автоматически сжимать видео через FFmpeg (CRF 28, 720p) для экономии места.
- Раскладывать лекции по папкам: `Дата / Зал / Время - Спикер - Тема.mp4`.

## Требования

- Python 3.10+
- FFmpeg (обязательно)
- Google Chrome (должен быть установлен в системе)

## Установка

1. **Клонировать репозиторий:**

    ```bash
    git clone https://github.com/pyramidheadshark/aij_2025_downloader.git
    cd aij_2025_downloader
    ```

2. **Установить FFmpeg (Критически важно):**
    - Скачайте полную сборку `ffmpeg.exe` (например, с [gyan.dev](https://www.gyan.dev/ffmpeg/builds/)).
    - **Положите файл `ffmpeg.exe` в корень папки этого проекта** (рядом с `main.py`).

3. **Установить зависимости Python:**

    ```bash
    pip install -r requirements.txt
    ```

4. **Установить драйверы браузера:**

    ```bash
    playwright install chromium
    ```

5. **Данные:**
    - Файл расписания должен лежать в `data/schedule.json` (уже включен в репозиторий).
